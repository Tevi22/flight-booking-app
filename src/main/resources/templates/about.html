About TevinAir

TevinAir is a fictional air travel booking web application developed by Tevin Davis for the IT 634 Distributed Application Development course at Southern New Hampshire University.

This application was built as part of Milestone Two, demonstrating the use of a custom RESTful API for searching flights, and showcasing integration of the Spring Boot MVC framework, Thymeleaf templating engine, and modular backend components. It includes unit testing, clear separation of concerns, and detailed in-code documentation.

  • Developer: Tevin Davis
  • Backend: Spring Boot, Java 17
  • Frontend: Thymeleaf, HTML5, Bootstrap 5
  • Milestone Version: Milestone Two (REST API, Search, MVC, and Testing)
  • Key Features:
    • RESTful search endpoint for flight data
    • Mock data service layer for flight listings
    • Flight search and results pages with form validation
    • Comprehensive unit tests for controllers
    • Reusable Thymeleaf fragments for consistent layout
  • Planned Enhancements:
    • User registration and login for booking
    • Support for round-trip bookings
    • Passenger age and class filters
    • Integration with a real flight API (e.g., SkyPicker)
